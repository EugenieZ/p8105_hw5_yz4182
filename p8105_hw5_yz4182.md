There are two cases to address:

Create missing data function

``` r
fill_in_missing = function(vector) {
  
  if (is.numeric(vector)) {
    
    vector =
      vector %>% 
      replace_na(mean(vector[!is.na(vector)]))
    
  }
  
  if (is.character(vector)) {
   
    vector = 
      vector %>% 
      replace_na("virginica")
    
  }
  
  return(vector)
}
```
